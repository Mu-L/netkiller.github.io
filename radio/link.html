3.4. Link Establishment(建立IP链路)

准备工作

主站点IP地址 192.168.6.1

节点点IP地址 192.168.6.2 一次类推

我们将中继的一时隙用作IP链路连接，二时隙用作本地通话，这样能做到两个时隙互不影响。

3.4.1. Master(主站设置)

首先设置主站点名称与Radio ID, Site ID

设置主站IP地址，注意Master IP为空，我们使用默认的50000端口，注意是UDP协议

设置频道，色吗12, IP Site Connect 选择1时隙，如果你想让两个时隙都工作可以选择 Slot 1 and Slot 2。然后填写RX频率，Offset 是差频-5，然后点击“Copy”按钮

最后将配置写入中继

3.4.2. Peer(从站设置)

设置从站点名称与Radio ID, Site ID 每个ID都不能重复，需要你建站的时候规划好，依次加一即可

从站需要填写Master IP, 然后设置从站的IP地址

从站是可以使用DHCP获取IP地址

设置频道
