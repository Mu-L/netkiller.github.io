10.3.4. 数据类型

10.3.4.1. 显示数据类型

			
import numpy

numpy.random.seed(1)

10.3.4.2. 转换数据类型

			
import numpy

numpy.random.seed(1)
matrix = numpy.random.randint(0,255,(5,6,3))
print(matrix.dtype)

matrix1 = matrix.astype(np.int8)
print(matrix1.dtype)
