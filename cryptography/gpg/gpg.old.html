6.11.4. Exporting keys (导出key)

6.11.4.1. --export export keys

				
gpg --export -a

6.11.4.2. -o, --output use as output file

				
$ gpg --export -a -o test.asc
$ gpg --output yourname.asc --export -a

6.11.5. Importing keys (导入key)

--import import/merge keys

			
neo@neo-laptop:~$ gpg --import 409.asc
