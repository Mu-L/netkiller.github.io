24.7.1. 安装模板

安装模板

			
npm install ejs 		
			
			
			
var express = require('express');
var app = express();

app.use(express.static('public'));
app.set("view engine","ejs");
app.set('views', __dirname + '/views');

24.7.2. 配置模板

24.7.2.1. 闭合标签

ejs 里，默认的闭合标记是 <% ... %> ，我们也可以定义自己的标签。例如： 

				

app.set("view options",{  
   "open":"{{",  
   "close":"}}"  
});
