1.1.1. Almalinux / RockyLinux

安装 Java 1.8 JRE

		
dnf install java-1.8.0-openjdk

安装 Java 1.8 JDK

		
dnf install java-1.8.0-openjdk-devel

当我们安装 maven 的时候会自动安装 Java 11，需要使用下面方法切换默认Java版本

		
[root@cloud ~]# alternatives --config java
