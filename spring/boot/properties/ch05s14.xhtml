Map类型

Map类型在properties和yaml中的标准配置方式如下：

		
properties格式：		
netkiller.key=value

yaml格式：
netkiller:
  key: value
  
举例：
user:
  name: neo
  gender: male
  age: 30

注意：如果Map类型的key包含字母数字和-以外的字符，需要用[]括起来，比如：

		
user:
  name:
    '[first.name]': neo
    '[last#name]': chen
