info 配置信息

返回 application.properties 文件中定义的 info 配置信息，如:

		
# info端点信息配置
info.app.name=spring-boot-example
info.app.version=v1.0.0		
		
		

		
neo@MacBook-Pro ~ % curl -s http://www.netkiller.cn:8080/actuator/info | jq   
{
  "app": {
    "name": "spring-boot-example",
    "version": "v1.0.0"
  }
}
