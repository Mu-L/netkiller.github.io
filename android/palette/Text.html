7.5.7. 禁止滚动条

临时禁止滚动条，让滚动失效

OnTouchListener 返回 true 禁止滚动

	    
        binding.recyclerView.setOnTouchListener(new View.OnTouchListener() {
            @Override
            public boolean onTouch(View view, MotionEvent motionEvent) {
                return true;
            }
        });

OnTouchListener 返回 false 恢复滚动

	    
        binding.recyclerView.setOnTouchListener(new View.OnTouchListener() {
            @Override
            public boolean onTouch(View view, MotionEvent motionEvent) {
                return false;
            }
        });
