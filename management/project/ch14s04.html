14.4. 岗位描述怎么写？

14.4.1. 产品经理

			
职责描述：
1. 负责公司环保交易平台网站和手机APP设计，商业模式的可行性分析
2. 需求调研、整理与分析，提出相关需求的改进意见
3. 将用户需求转化为产品原型图和需求文档，完成交互设计，界面设计，创意设计。
4. 与设计，开发，测试团队对接，积极沟通，完成产品测试与交付验收
5. 负责编写产品使用说明书，产品宣讲与使用培训

任职要求：
1. 具有三年以上产品经理工作经验，有B2B，SaaS 平台相关工作经验；
3. 熟练使用 Axure RP, Sektch，Visco，Adobe 等软件，能够提供高质量的产品原型、流程图、线框图等清晰的表达设计方案；
4. 精通电商系统，供求信息，招标/采购，合同签章，在线支付，物流等模块的前后台设计
5. 对需求分析、数据分析等有独到的见解，善于发掘有价值的业务数据
6. 具备较强的文案能力及沟通、交流、协调能力，具有良好的团队合作精神，自学能力强。


优先考虑（加分项）：
1. 供应链金融系统的设计
2. 工作涉及过相机，地图，定位，二维码，NFC，加速度传感器，蓝牙通信
3. 物联网，硬件产品，边缘计算
4. 区块链
5. 打通三屏的经验（电脑，手机，平板电脑）

14.4.2. 高级运维

					
岗位职责：
1.全面负责公司运维项目的系统升级、扩容需求与资源落实，配合开发需求，测试、调整运维平台；
2.负责IT运维相关流程的规划、设计、推行、实施和持续改进；
3.全面负责网络以及服务器、安全监控、检查并消除安全漏洞、系统性能管理和优化、数据高可用性、容灾和故障恢复；；
4.大用户量下高性能服务器系统部署方案的制定及实施；规划并制定符合未来业务发展规模的服务架构计划（运维架构）；
5.负责运维文档的规范与管理工作。
招聘要求：
1、统招大专及以上学历，互联网运维工作五年以上；
2、精通Linux/Unix系统及原理，了解网络技术，熟悉TCP/IP协议工作原理，对路由器，交换机，防火墙的配置有一定的了解；
3、精通阿里云：云监控、WAF防火墙、SLB负载均衡、堡垒机、阿里云 Kubernetes 、RocketMQ、RDS、ES、Redis、DTS、MSE配置注册中心、容器镜像库
4、精通  Docker、能够在本地独立搭建 Kubernetes 集群；
5、精通 Prometheus，Open Falcon等监控工具，熟练使用Shell、Python、能开发 Prometheus exporter 探针
6、熟悉常见应用的部署及优化，如 Nginx/ Openresty /Tomcat/Gitlab/ELK/Jenkins/MySQL/Redis/MongoDB/MQ 等；
7、熟悉SpringCloud alibaba Nacos 维服务体系，在没有开发协助下独立部署。
8、善于思考，有很强的问题分析能力和推进解决的能力

14.4.3. 服务器与硬件设备

raid原理

				 
高级运维工程师/运维经理 30K～40K
岗位职责：
1.全面负责公司运维项目的系统升级、扩容需求与资源落实，配合开发需求，测试、调整运维平台；
2.负责IT运维手册及文档撰写，相关流程的规划、设计、制定、推行、实施和持续改进；
3.负责网络以及服务器、安全监控、检查并消除安全漏洞、系统性能管理和优化、数据库高可用性、容灾和故障恢复；
4.大用户量下高性能服务器系统部署方案的制定及实施；规划并制定符合未来业务发展规模的服务架构计划（运维架构）；
招聘要求：
1、互联网运维工作5年以上；
2、必须精通阿里云：云监控、WAF防火墙、SLB负载均衡、SLS日志服务，堡垒机、阿里云 Kubernetes 、RocketMQ、RDS、ES、Redis、DTS、MSE配置注册中心、容器镜像库3、精通Linux/Unix系统及原理，了解网络技术，熟悉TCP/IP协议工作原理，对路由器，交换机，防火墙的配置有一定的了解；
4、精通  Docker、能够在本地独立搭建 Kubernetes 集群；
5、精通 Prometheus 等监控工具，熟练使用Shell、Python、能开发 Prometheus exporter 探针
6、熟悉常见应用的部署及优化，如 Nginx/ Openresty /Tomcat/Gitlab/ELK/Jenkins/MySQL/Redis/MongoDB/MQ 等；
7、熟悉SpringCloud alibaba Nacos 微服务体系，能够在没有开发协助下独立部署。
8、善于思考，有很强的问题分析能力和解决问题的能力

14.4.4. 高级 Java 软件工程师

岗位职责：

				 
1、参与项目的需求讨论，项目功能模块设计；
2、负责项目核心模块开发及优化改进；
3、能够及时发现项目存在的问题加以改进，并向项目经理汇报，以确保项目的顺利进行；
4、对软件系统及模块进行测试工作，修改BUG；
5、攻克产品技术难点，进行技术研究性工作。

任职资格：

				 
1. 计算机相关专业，至少5年基于Java技术的软件开发经验，基础扎实，开发思路简洁严谨；
2. 精通SpringBoot，SpringCloud 微服务，Webflux Restful开发；
3. 熟练掌握H5 Web开发，Vue.js 框架
4. 精通MySQL数据库，能够写出高性能SQL查询语句，本项目采用 JPA 和 JDBC 方式开发；
5. 在项目中使用过 Oauth2, JPA, Spring Security, Redis , Kafka
6. 执行能力强，沟通能力强，协调及组织能力强，有责任心和工作担当；
7. 对新技术感兴趣，能主动并且敢于攻克技术难关，积极研发的能力。

有以下任一经验者优先：

				 
1. 对缓存、分布式、负载均衡有经验者优先；
2. 在微服务方面有相关实践经验者优先；
3. 有搜索引擎相关经验者优先；经纬度范围搜索经验者
4. 有推荐引擎经验者优先

14.4.5. Flutter App 开发工程师

			 
岗位职责：
负责开发手机APP，安卓手持终端，iPad 平板应用

任职资格：
1. 两年 flutter 开发经验
2. 精通 Dart 语言，熟练 iOS, Android 应用开发
3. 会使用Flutter布局复杂页面，包括手机，平板，电脑等。
4. 精通基于 Oauth2 认证的 Restful 接口调用
5. 能够熟练封装组件以及插件开发

有以下经验者优先：
1. Java 安卓开发
2. iOS 开发

14.4.6. 测试工程师

岗位职责:

				 
1.对负责手机和平板APP测试；
2.参与产品的需求评审工作；
3.准备测试环境和测试用例，执行测试，提交缺陷，并跟进修复验证；
4.能够按照上线流程执行上线，做好线上检查，跟进用户反馈情况；
5.按照部门流程规范完成项目文档更新和保存。

任职要求:

				 
1. 至少3年以上互联网测试工作经验；
2. 具备基础的移动测试知识，熟练掌握测试用例设计和app测试方法；
3. 熟练使用主流测试工具；例如jmeter、fiddler、adb、android studio等；
4. 对测试有浓厚的兴趣，有上进心，善于沟通，逻辑性强，学习能力强，且具有创新进取精神。
